# Приложение для проведения экзамена
***
### Проект написан разработчиком https://github.com/NickShkril
***
### Данное приложение разработано для автоматизации проведения экзамена

### Стек технологий
***
**В проекте используются**:

* Backend:
    - Java 11
    - Maven
    - Spring Boot
    - Spring Web
    - Spring Data
    - Spring JPA
    - GIT
    - REST
* Frontend:
    - Swagger 
* SQL:
    - PostgreSQL
    - Liquibase
